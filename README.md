# Create dataset putting noise in it :file_folder:

- If (x1 – 0.5)2 + (x2 – 0.5)2 <0.16, then (x1,x2) is classified in the category C1,

- If (x1 + 0.5)2 + (x2 + 0.5)2 <0.16, then (x1,x2) is classified in the category C1,

- If (x1 – 0.5)2 + (x2 + 0.5)2 <0.16, then (x1,x2) is classified in the category C2,

- If (x1 + 0.5)2 + (x2 – 0.5)2 <0.16, then (x1,x2) is classified in the category C2,
